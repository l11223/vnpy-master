# Mac版本界面说明

## ✅ 是的！Mac版本界面完全一样

打包后的Mac版本会显示**完全相同的GUI界面**，包括：

### 📊 界面组成（和原版一样）

1. **顶部菜单栏**
   - 系统、功能、配置、帮助
   - 所有功能菜单

2. **左侧交易面板**
   - 交易所选择（SHFE、DCE、CFFEX等）
   - 代码输入（合约代码）
   - 方向选择（多/空）
   - 开平选择（开仓/平仓）
   - 类型选择（限价/市价等）
   - 价格和数量输入
   - 委托按钮、全撤按钮

3. **右侧数据面板**
   - **行情**：实时行情数据表格
   - **委托**：活动委托、历史委托
   - **成交**：成交记录
   - **持仓**：当前持仓
   - **账户**：账户信息

4. **其他功能模块**
   - CTA策略
   - 数据管理
   - 回测
   - 等等...

## 🎨 界面特点

- ✅ **深色主题**：使用qdarkstyle，和原版一样
- ✅ **完整功能**：所有原版功能都保留
- ✅ **Mac适配**：在Mac上完美显示
- ✅ **响应式布局**：窗口可调整大小

## 🚀 启动方式

### 方式1：UI模式（图形界面）

```bash
python3 启动量化系统.py
```

或者打包后的App：
```bash
双击 VNPY.app
```

### 方式2：无UI模式（后台运行）

```bash
python3 启动量化系统.py --no-ui
```
