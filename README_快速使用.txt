================================================================================
一、快速启动（3种方式）
================================================================================

方式1: 测试量化功能（推荐先运行这个）
    python3 测试量化功能.py

方式2: 启动完整系统（UI界面）
    python3 启动量化系统.py

方式3: 启动完整系统（后台运行，无界面）
    python3 启动量化系统.py --no-ui

================================================================================
二、如果遇到依赖错误
================================================================================

错误: ModuleNotFoundError: No module named 'loguru'

解决: 
    pip install loguru tzlocal

如果需要UI界面:
    pip install PySide6 qdarkstyle

如果需要数据分析:
    pip install matplotlib numpy pandas
