# 快速启动指南

## 如果系统卡住或启动失败

### 方法1: 安装所有依赖（推荐）

```bash
# 运行安装脚本
bash 安装依赖.sh

# 或者手动安装
pip install loguru tzlocal TA-Lib qdarkstyle PySide6 vnpy
```

### 方法2: 使用无UI模式（最简单）

如果UI模式启动失败，使用无UI模式：

```bash
python3 启动量化系统.py --no-ui
```

### 方法3: 直接测试功能（不需要UI）

```bash
python3 测试量化功能.py
```
